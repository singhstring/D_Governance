# D_GOVERNANCE

> Blockchain-based decentralized government document storage system as NFTs (2017)
> Ethereum Dapp Project in Solidity ,VueJS with IPFS


# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
